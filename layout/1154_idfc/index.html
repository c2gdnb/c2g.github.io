Dedicated (выделенный сервер)

Вид хостинга, при котором клиенту целиком предоставляется отдельная физическая машина (в противоположность виртуальному хостингу). Обычно используется для запуска приложений, которые не могут сосуществовать на одном сервере с другими проектами или имеют повышенные требования к ресурсам. Для крупных проектов: интернет магазинов (от 50 000 товаров) с интеграциями, веб-сервисы, корпоративные порталы, индивидуальные проекты.
